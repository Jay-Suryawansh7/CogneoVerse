## ÔøΩ Community Guidelines

At Cogneverse, we believe in **Innovation through Collaboration**. To maintain a healthy and productive environment, we ask all contributors to adhere to our core values:

### 1. Innovation First
We push boundaries. Don't be afraid to propose radical ideas or experiment with new technologies. We value "Why not?" over "It's never been done."

### 2. Collaborative Spirit
Great things happen when brilliant minds work together.
-   **Be Respectful:** Treat every contributor with kindness and respect.
-   **Share Knowledge:** Mentorship is key. If you know something, teach it.
-   **Constructive Feedback:** Critique ideas, not people. Help others improve their work.

### 3. Ship & Iterate
Perfection is the enemy of progress. We believe in rapid prototyping, shipping early, and learning from user feedback.

### 4. Global Impact
Our code touches lives across continents. Always consider accessibility, localization, and inclusivity in your designs and implementations.

## ü§ù How to Contribute

We welcome contributions from everyone‚Äîwhether you're a seasoned engineer, a design student, or a first-time open-source contributor.

### What We're Looking For
-   **Code:** Bug fixes, performance improvements, new features (React/Next.js).
-   **Design:** UI/UX improvements, accessibility audits, animations.
-   **Documentation:** Tutorials, guides, and translation of docs.
-   **Ideas:** Feature proposals and architectural discussions.

### Contribution Workflow

1.  **Find an Issue:** Check our [Issues](https://github.com/Jay-Suryawansh7/CogneoVerse/issues) page for "good first issue" or "help wanted" tags.
2.  **Fork & Branch:**
    -   Fork the repository.
    -   Create a new branch: `git checkout -b feature/your-feature-name`.
3.  **Make Changes:** Write clean, documented code following our style guide (Tailwind CSS, TypeScript).
4.  **Test:** Ensure your changes don't break existing functionality.
    -   Run `npm run dev` to test locally.
5.  **Submit a Pull Request (PR):**
    -   Push to your fork: `git push origin feature/your-feature-name`.
    -   Open a PR against the `main` branch.
    -   Describe your changes and link any related issues.

## ÔøΩ Tech Stack & Setup

For developers looking to jump into the codebase, here is our core technology stack:

-   **Framework**: [Next.js 14+](https://nextjs.org/) (App Router)
-   **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
-   **Animation**: [Framer Motion](https://www.framer.com/motion/) & CSS Keyframes
-   **Icons**: [Lucide React](https://lucide.dev/)
-   **Language**: TypeScript

### Quick Start

```bash
# Clone the repository
git clone https://github.com/Jay-Suryawansh7/CogneoVerse.git

# Navigate to the app directory
cd CogneoVerse/cogneverse-app

# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to join the universe.
