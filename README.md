# CogneoVerse

An open innovation ecosystem where engineers, designers, and AI builders collaborate to turn ideas into real-world technology.

## Getting Started

```bash
cd cogneverse-app
npm install
npm run dev
```

Visit http://localhost:3000 to see the site.

## Tech Stack
- Next.js 16
- TypeScript
- Tailwind CSS v4
- Framer Motion
- shadcn/ui
